## Finding Companies with Bugs (LinkedIn Strategy)

### Method 1: Manual Testing

1. **Find companies:**
   - Search LinkedIn: "HR tech companies", "EdTech startups"
   - Find their websites
   - Test their apps/websites

2. **Find bugs:**
   - Broken links
   - Slow loading
   - UI/UX issues
   - Security issues (responsibly disclose)
   - API errors

3. **Document and email:**
   - Screenshot the bug
   - Explain the impact
   - Offer to fix: "I can fix this in 2 days for $X"

### Method 2: Use Tools

**Tools to find companies:**
- **BuiltWith** - See what tech companies use
- **Wappalyzer** - Browser extension to see tech stack
- **SimilarWeb** - Competitor analysis
- **Crunchbase** - Company database

**Tools to find bugs:**
- **Google PageSpeed Insights** - Performance issues
- **WAVE** - Accessibility issues
- **SSL Labs** - Security issues
- **GTmetrix** - Performance problems

### Method 3: LinkedIn Search Strategy

**Search queries:**
- "CTO at [company type]"
- "Founder at [startup type]"
- "Engineering Manager at [tech company]"

**Message template:**
```
Hi [Name],

I'm a developer who specializes in [your skills]. I noticed [company] is in [industry].

I built [your solution] that could help [specific problem in their industry].

Would you be open to a quick chat to see if this could be useful?

Best,
[Your Name]
```
